# Conversation Workbench for ChatGPT

🧩 一个专注于优化长篇 ChatGPT 对话体验的浏览器扩展，通过整理对话降低卡顿，并提供时间线导航、搜索、导出、提示词管理与偏好设置能力。  

🧩 A browser extension focused on improving long ChatGPT conversations by reducing lag through conversation cleanup, with timeline navigation, search, export, prompt management, and preference settings.

---

## ✨ 简介 | Overview

**Conversation Workbench for ChatGPT** 是一个面向长对话场景的效率工具，重点解决以下问题：

- 长对话越来越卡，阅读和滚动体验变差
- 对话变长后，查找历史内容很困难
- 缺少更直观的导航方式，难以快速定位上下文
- 导出、提示词管理和偏好设置不够集中、不够顺手

本项目通过 **整理对话、时间线导航、搜索、导出、提示词库与偏好设置** 等能力，帮助你更轻松地处理长篇 ChatGPT 对话。✨

**Conversation Workbench for ChatGPT** is a productivity toolkit built for long conversations. It is designed to address common pain points such as:

- increasing lag in long chats
- difficulty finding earlier messages
- lack of intuitive navigation across long threads
- fragmented workflows for export, prompt management, and preferences

With **conversation cleanup, timeline navigation, search, export, prompt library, and preference settings**, this project helps make long ChatGPT conversations easier to manage. ✨
